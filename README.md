# 🍴 Flutter Recipe App | Firebase Firestore Powered

A complete cross-platform recipe app built with Flutter & Firebase Firestore, designed for food lovers to explore, save, and manage recipes from around the world.

This app was created to solve the common problem of scattered recipes online — giving users a clean, organized, and personalized way to discover new dishes, save favorites, and even contribute their own recipes to the community.

---

## ✨ Key Features

- 🌍 **Country-Based Recipe Search**  
  Browse recipes by country/region to discover authentic cuisines from around the world.

- 📖 **Detailed Recipe View**  
  Every recipe includes descreption, step-by-step instructions, ingredient lists, prepare time, and cooking time.

- ❤️ **Favorites & Bookmarks**  
  Save your go-to recipes to a personal favorites list, synced across all your devices via Firebase.

- ➕ **Add/Edit/Delete Recipes**  
  Full CRUD functionality to create your own recipes, edit existing ones, or remove them anytime.

- 🔍 **Smart Search**  
  Find recipes by name, ingredient, or cuisine type with instant results.

- 🔐 **Firebase Authentication**  
  Secure user sign-up/login to keep your recipes and favorites private.

- ☁️ **Firebase Firestore Backend**  
  All recipe data is stored in a fast, scalable cloud database, ensuring real-time updates.

- 🖼️ **Direct URL Image Loading**  
  Recipe images are loaded directly from public URLs (no Firebase Storage dependency) for faster and simpler setup.

- 📱 **Cross-Platform Support**  
  Runs smoothly on Android, iOS, Web, Windows, macOS, and Linux from a single codebase.
